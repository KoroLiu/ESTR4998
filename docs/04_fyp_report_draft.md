## 关于临时冲击 (Temporary Impact) 的影响
根据测试，当调大 `eta2` 时，智能体为了避免高额的瞬时滑点，会显著将交易平摊到整个时间轴上，这与理论 PDE 解的预测完全一致。图表已在 `04_Evaluation` 中生成。

## RL 方案的优势讨论
虽然 PDE 可以在特定二次冲击设定下得到数值解 (如论文 Eq 40, 44)，但 RL (PPO) 方案提供了一种 Model-free 的解法。这意味着如果我们未来把冲击函数 $g(v)$ 换成不可微的经验公式，或者引入真实的 Limit Order Book 深度数据，RL 依然可以直接适用，这是传统解析方法难以做到的。
